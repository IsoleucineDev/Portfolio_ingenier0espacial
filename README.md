# Manu's Music Portfolio

Welcome to Manu's Music Portfolio! This platform showcases musical talent using cutting-edge technology, combining both artistry and modern web development.

## Tech Stack

This portfolio is built with a powerful modern tech stack:

- **TypeScript (92.8%)**: Provides robust, type-safe coding, ensuring a seamless and error-free experience.
- **Next.js (15)**: A React framework that enables server-side rendering and static site generation for optimal performance.
- **React (19)**: The core library for building interactive user interfaces.
- **Turbopack**: A next-generation packer optimized for speed and performance.
- **Tailwind CSS**: A utility-first CSS framework that enables rapid styling and customization of the UI.
- **SoundCloud Integration**: Showcase your music tracks directly from SoundCloud for easy access and sharing.
- **Animations**: Enhance user experience with smooth and attractive animations.

## Features

- Interactive music player with integrated SoundCloud tracks.
- Responsive design that adapts to any device.
- Sleek and modern aesthetic, emphasizing usability and accessibility.
