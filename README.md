## Instalación y uso

Para clonar y ejecutar localmente este proyecto:

```bash
git clone https://github.com/IsoleucineDev/Portfolio_ingenier0espacial.git
cd Portfolio_ingenier0espacial
npm install
npm run dev
```

Accede en tu navegador a http://localhost:3000 para ver el portafolio en acción.
